🧩 六大核心模块详解
📦
商品信息解析 Agent
Product Info Parser
自动抓取多平台店铺商品数据，通过长链推理拆解商品核心属性与上下架要求，智能识别平台差异规则。
多平台数据抓取 长链推理 规则解析
⚖️
合规检测 Agent
Compliance Detector
基于最新广告法与平台规则，扫描商品标题、主图、详情页全内容违规风险，生成优化方案并追踪整改。
广告法检测 平台规则 风险预警
🔄
智能上下架调度 Agent
Smart Scheduler
通过长链推理生成最优上下架策略，自动执行多平台批量操作，支持库存阈值触发自动上下架。
批量操作 策略优化 阈值触发
📊
库存同步 Agent
Inventory Sync
打通 ERP 与多平台库存数据，实时监控各平台库存状态，库存低于阈值自动下架，补足后自动上架，超卖风险清零。
ERP对接 实时同步 超卖防护
✨
商品优化 Agent
Product Optimizer
基于实时流量数据，自动优化商品标题、主图、卖点文案，结合搜索排名算法持续提升商品曝光与转化率。
标题优化 流量分析 转化提升
📋
运营复盘 Agent
Operations Reviewer
汇总全流程运营数据，生成多维度复盘报告，反向迭代优化上下架策略，形成完整自闭环，持续提升运营效果。
数据复盘 策略迭代 闭环优化
